# 🐘 Estudos de SGBD e Conexão com PDO

Este repositório contém implementações práticas desenvolvidas durante meus estudos acadêmicos sobre **Sistemas de Gerenciamento de Banco de Dados (SGBD)**. O objetivo principal é consolidar o conhecimento em integração entre PHP e diferentes motores de banco de dados de forma segura.

## 🚀 Tecnologias Utilizadas

* **PHP 8.x**
* **PDO (PHP Data Objects):** Utilizado pela sua camada de abstração que permite trocar de banco de dados com facilidade e segurança contra SQL Injection.
* **MySQL:** Exemplos de conexão e manipulação.
* **PostgreSQL:** Implementação de drivers para o ecossistema Postgres.

## 📁 Estrutura do Repositório

O projeto está organizado por pastas focadas em cada tipo de conexão:

* `SGBD-PDO/`: Conceitos fundamentais e padrões de projeto para abstração de dados.
* `conncection-mysql/`: Configurações específicas, drivers e tratamento de erros para MySQL.
* `Connection-postgre/`: Implementação da classe de conexão voltada ao PostgreSQL.
